## 特别说明
2017年7月14日开始更新的规则都是基于[git@lhie1](https://github.com/lhie1/Surge)转换并添加一些过滤规则而成的Wingy规则。

## 使用说明

使用环境：Wingy 正式版或TestFlight版  
规则有过滤广告、代理、直连  
支持多线路，请参考修改  
[自定义配置的详细说明](https://github.com/hellowingy/wingy-announcement/blob/master/CONFIG.md)
